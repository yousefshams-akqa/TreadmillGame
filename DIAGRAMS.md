## 4. Game Loop Data Flow

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           GAME LOOP                                      │
└─────────────────────────────────────────────────────────────────────────┘

                    ┌───────────────────────────┐
                    │      ACCELEROMETER        │
                    │     (50 samples/sec)      │
                    └─────────────┬─────────────┘
                                  │
                                  ▼
                    ┌───────────────────────────┐
                    │    StepsRepository        │
                    │                           │
                    │  1. Detect step pattern   │
                    │  2. Calculate speed       │
                    │  3. Calculate distance    │
                    └─────────────┬─────────────┘
                                  │
                                  │ AsyncStream<StepsResult>
                                  ▼
┌─────────────────────────────────────────────────────────────────────────┐
│                         GameController                                   │
│                                                                          │
│  for await stepsResult in stepsRepo.stepsStream {                       │
│                                                                          │
│      ┌──────────────┐    ┌──────────────┐    ┌──────────────┐          │
│      │ Update UI    │    │ Enemy Tick   │    │ Check        │          │
│      │ Properties   │    │ (10Hz)       │    │ Win/Lose     │          │
│      │              │    │              │    │              │          │
│      │ - steps      │    │ - spawn      │    │ - stepGoal   │          │
│      │ - speed      │    │ - closeness  │    │ - caught     │          │
│      │ - distance   │    │ - catch      │    │              │          │
│      └──────────────┘    └──────────────┘    └──────────────┘          │
│             │                   │                   │                   │
│             ▼                   ▼                   ▼                   │
│      ┌──────────────┐    ┌──────────────┐    ┌──────────────┐          │
│      │ Haptics      │    │ Sound FX     │    │ TTS Voice    │          │
│      │ Feedback     │    │              │    │ Cues         │          │
│      └──────────────┘    └──────────────┘    └──────────────┘          │
│  }                                                                       │
└─────────────────────────────────────────────────────────────────────────┘
                                  │
                                  │ @Published properties
                                  ▼
                    ┌───────────────────────────┐
                    │       GameScreen          │
                    │                           │
                    │  SwiftUI automatically    │
                    │  re-renders on changes    │
                    └───────────────────────────┘
```

## 5. Enemy Lifecycle

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        ENEMY LIFECYCLE                                   │
└─────────────────────────────────────────────────────────────────────────┘

 Player Steps:  0      50     100    150    200    250    300    350
                │       │       │       │       │       │       │
                ▼       ▼       ▼       ▼       ▼       ▼       ▼
 ───────────────●───────●───────●───────●───────●───────●───────●──────▶

                        │
                        │ spawnAtStep: 50
                        ▼
                ┌───────────────┐
                │ ENEMY SPAWNS  │
                │               │
                │ startSound()  │
                │ startHaptics()│
                └───────┬───────┘
                        │
                        ▼
        ┌───────────────────────────────────────┐
        │           ENEMY CHASING               │
        │                                       │
        │  Enemy steps = time × stepsPerSecond  │
        │                                       │
        │  ┌─────────────────────────────────┐  │
        │  │ Closeness = 1 - (gap / 25)      │  │
        │  │                                 │  │
        │  │  0.0 ─────────────────────▶ 1.0 │  │
        │  │  Safe              Danger       │  │
        │  └─────────────────────────────────┘  │
        │                                       │
        │  Haptics & Audio scale with closeness │
        └───────────────────┬───────────────────┘
                            │
              ┌─────────────┴─────────────┐
              │                           │
              ▼                           ▼
    ┌─────────────────┐         ┌─────────────────┐
    │  ENEMY CATCHES  │         │  ENEMY GIVES UP │
    │    PLAYER       │         │                 │
    │                 │         │ maxSteps reached│
    │  ⚠️ GAME OVER   │         │                 │
    │ (after grace s) │         │ Enemy destroyed │
    │ stepGoalReached │         │ Wait for next   │
    │    = false      │         │ spawn trigger   │
    └─────────────────┘         └─────────────────┘
```

┌─────────────────────────────────────────────────────────────────────────┐
│                     LEVEL CONFIGURATION                                  │

│   JSON File     │      chase.json / tutorial.json / survival.json

         │
         │ FetchLevelsRepository.loadBundledLevel()

│     Level       │      Decoded Swift model
│                 │
│ - name          │
│ - stepGoal      │
│ - enemies       │
│ - audio         │
│ - ttsMessages   │
└────────┬────────┘
         │
         │ + UserSettings (strideLength, haptics, etc.)
         ▼
┌─────────────────┐
│ GameSessionConf │      Combined configuration
│                 │
│ - level         │
│ - userSettings  │
│ - effectiveVals │
└────────┬────────┘
         │
         │ GameController(config:)
         ▼
┌─────────────────┐
│ GameController  │      Ready to play
│                 │
│ Repositories    │

│ level settings  │
